== Description ==

Allow customers to collect orders from RedBox Lockers, which are available around the city 24/7.

== Installation ==

1. Upload folder `Redbox pickup` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Setting key of your store in Redbox at submenu of WooCommerce
4. Create new shipping method with type "Redbox pickup" in shipping zone
